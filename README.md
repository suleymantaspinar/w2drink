# w2drink

w2drink is a cocktail assistant which you can get cocktail recipes.

# Installation

Backend Setup (without Docker):   

```
cd backend
npm install
npm run start
```

Frontend Setup (without Docker):   
```
cd frontend
npm install
npm run start
```
or

Use docker compose:  
```
docker-compose up
```
